# 相对估值

相对估值是通过与可比公司或历史估值水平进行比较来确定目标公司价值的方法。

## 相对估值原理
### 基本概念
相对估值的核心思想是"相似公司应该有相似的估值"，通过市场定价的相对比较来判断投资价值。

### 理论基础
- **有效市场假说**：市场总体上是有效的
- **替代原则**：相似资产可以相互替代
- **相对价值**：价值是相对比较的结果

## 常用相对估值指标
### 市盈率（P/E）
**计算公式：**
```
市盈率 = 股价 / 每股收益
```

**类型：**
- 静态市盈率（TTM）
- 动态市盈率（预期）
- 滚动市盈率

**适用情况：**
- 盈利稳定的企业
- 成熟型企业
- 同行业比较

**局限性：**
- 亏损企业无法使用
- 周期性企业误判风险
- 会计政策差异影响

### 市净率（P/B）
**计算公式：**
```
市净率 = 股价 / 每股净资产
```

**适用情况：**
- 金融企业
- 重资产企业
- 周期性企业

**优势：**
- 净资产相对稳定
- 适合资产密集型行业
- 波动相对较小

**注意事项：**
- 净资产质量差异
- 无形资产影响
- 会计政策差异

### 市销率（P/S）
**计算公式：**
```
市销率 = 总市值 / 营业收入
```

**适用情况：**
- 成长型企业
- 亏损企业
- 周期性企业底部

**优势：**
- 收入相对稳定
- 不受盈利波动影响
- 适合不同会计准则比较

**局限性：**
- 忽略盈利能力
- 不适合利润率差异大的行业

### EV/EBITDA
**计算公式：**
```
EV/EBITDA = 企业价值 / 息税折旧摊销前利润
```

**企业价值（EV）计算：**
```
EV = 市值 + 总负债 - 现金
```

**适用情况：**
- 跨行业比较
- 资本结构差异大的企业
- 国际比较

**优势：**
- 消除资本结构影响
- 消除折旧摊销影响
- 适合并购估值

## 可比公司选择
### 选择标准
**行业相似性：**
- 主营业务相同
- 商业模式相似
- 客户群体重叠

**规模相似性：**
- 营业收入规模相近
- 资产规模相近
- 市场地位相似

**成长性相似：**
- 历史增长率相近
- 未来增长预期相似
- 生命周期阶段相近

**风险特征相似：**
- 财务风险水平
- 经营风险特征
- 市场风险暴露

### 可比公司数据库
**国内市场：**
- 同行业上市公司
- 科创板/创业板企业
- 新三板企业

**国际市场：**
- 海外同行业公司
- 新兴市场公司
- 发达市场公司

**数据库来源：**
- Wind数据库
- Bloomberg终端
- 各大券商研报

## 相对估值方法
### 算术平均法
**步骤：**
1. 选择可比公司
2. 计算各公司估值指标
3. 计算算术平均值
4. 应用到目标公司

**优缺点：**
- 简单易用
- 受极端值影响大
- 未考虑公司差异

### 加权平均法
**权重考虑：**
- 市值权重
- 收入权重
- 主观判断权重

**公式：**
```
加权平均估值 = Σ(估值i × 权重i)
```

### 回归分析法
**模型建立：**
```
估值 = a + b1×因素1 + b2×因素2 + ... + ε
```

**常见因素：**
- 盈利增长率
- ROE水平
- 负债率
- 市场份额

## 历史估值比较
### 历史估值区间
**分析内容：**
- 历史估值波动范围
- 估值中枢变化趋势
- 极端值出现原因

**应用价值：**
- 判断当前估值水平
- 识别估值回归机会
- 理解估值驱动因素

### 估值驱动因素分析
**基本面因素：**
- 盈利增长率
- 盈利质量
- 资产效率

**市场因素：**
- 流动性环境
- 风险偏好
- 行业景气度

**政策因素：**
- 监管政策变化
- 产业政策调整
- 宏观政策导向

## 跨市场估值比较
### 国际比较方法
**汇率调整：**
```
调整后估值 = 原估值 × 汇率因子
```

**会计准则调整：**
- IFRS vs GAAP vs 中国会计准则
- 会计政策差异调整
- 信息披露差异考虑

**市场特征调整：**
- 市场有效性差异
- 投资者结构差异
- 流动性差异

### A/H股比较
**溢价/折价分析：**
```
A/H溢价率 = (A股价格 - H股价格×汇率) / (H股价格×汇率)
```

**影响因素：**
- 市场分割
- 投资者结构
- 流动性差异
- 信息不对称

## 相对估值应用场景
### 价值投资
- 寻找被低估股票
- 同行业轮动
- 小市值价值发现

### 成长投资
- 成长性调整后的估值
- 增长潜力比较
- 行业龙头溢价

### 事件驱动
- 并购重组估值
- 重组对标分析
- 定增定价参考

## 相对估值局限性
### 市场有效性问题
- 市场情绪影响
- 泡沫和低估并存
- 市场非理性波动

### 可比公司选择困难
- 完全可比公司少
- 行业边界模糊
- 商业模式差异

### 估值指标适用性
- 不同行业适用指标不同
- 同一指标不同时期表现不同
- 单一指标局限性大

## 提高相对估值准确性
### 多指标结合
- 综合使用多个估值指标
- 指标间相互验证
- 行业特色指标应用

### 动态调整
- 定期更新可比公司
- 调整权重和参数
- 跟踪估值变化趋势

### 质量调整
- 盈利质量差异调整
- 成长性差异调整
- 风险特征调整

## 投资策略结合
### 绝对估值+相对估值
- 相对估值找机会
- 绝对估值验价值
- 综合判断投资价值

### 定量+定性分析
- 定量指标筛选
- 定性深度分析
- 综合投资决策

### 长期+短期视角
- 长期价值判断
- 短期交易机会
- 组合策略设计
