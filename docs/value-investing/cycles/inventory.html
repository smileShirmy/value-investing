库存周期 ​

库存周期是经济周期的重要组成部分，反映了企业库存水平的周期性变化对经济活动的影响。

库存周期理论 ​

库存周期通常持续3-4年，包括四个阶段：

  • 主动补库存：企业预期向好，主动增加库存
  • 被动补库存：需求下滑，库存被动积压
  • 主动去库存：企业预期悲观，主动减少库存
  • 被动去库存：需求回升，库存被动消化

四个阶段分析 ​

主动补库存（繁荣期） ​

特征：

  • 需求旺盛，销售增长
  • 企业对未来乐观
  • 主动增加原材料和产成品库存
  • 产能利用率提高

市场表现：

  • 原材料价格上涨
  • 工业产出增加
  • 企业收入利润双增长

投资机会：

  • 上游资源品股票
  • 中游制造业股票
  • 关注需求持续性和通胀压力

被动补库存（衰退初期） ​

特征：

  • 需求开始下滑
  • 库存被动积压
  • 产能利用率下降
  • 企业开始调整生产计划

市场表现：

  • 原材料价格见顶回落
  • 库存数据居高不下
  • 企业盈利增速放缓

投资策略：

  • 减持周期股
  • 关注防御性行业
  • 规避库存高企的企业

主动去库存（衰退期） ​

特征：

  • 需求疲软，销售下滑
  • 企业主动削减库存
  • 减少生产，裁员
  • 价格竞争加剧

市场表现：

  • 原材料价格大幅下跌
  • 工业产出收缩
  • 通货紧缩压力

投资机会：

  • 防御性行业
  • 现金类资产
  • 等待底部信号

被动去库存（复苏期） ​

特征：

  • 需求开始回升
  • 库存快速消化
  • 产能利用率提高
  • 企业重新开始生产

市场表现：

  • 原材料价格企稳回升
  • 工业产出恢复增长
  • 企业盈利改善

投资策略：

  • 提前布局周期股
  • 关注库存数据变化
  • 把握经济复苏机会

库存指标监控 ​

主要指标 ​

  • 工业企业产成品库存：反映库存水平变化
  • PMI库存指数：采购经理人调查的库存数据
  • 存货周转天数：企业库存管理效率
  • 原材料库存指数：上游库存变化

分析要点 ​

  • 关注库存同比变化趋势
  • 结合需求和产能分析
  • 注意库存结构（原材料vs产成品）
  • 考虑不同行业库存周期差异

行业差异 ​

上游资源行业 ​

  • 特点：库存周期波动大
  • 投资要点：关注价格趋势和供需关系

中游制造业 ​

  • 特点：库存管理更精细
  • 投资要点：关注产能利用率和库存周转

下游消费行业 ​

  • 特点：库存相对稳定
  • 投资要点：关注渠道库存和终端销售

投资应用 ​

如何利用库存周期 ​

  • 监控库存数据判断经济周期位置
  • 在被动去库存期提前布局
  • 在主动补库存期享受上涨行情
  • 在被动补库存期及时减仓

风险控制 ​

  • 库存周期存在不确定性
  • 政策干预可能改变周期节奏
  • 需要结合其他指标综合判断
  • 注意不同行业的周期性差异
