绝对估值 ​

绝对估值是通过分析企业的基本面来确定其内在价值的方法，不依赖于市场价格的比较。

现金流折现模型（DCF） ​

基本原理 ​

DCF模型的核心思想是企业的价值等于其未来现金流的现值之和：

企业价值 = ∑[FCFt / (1+r)^t] + 终值

其中：

  • FCFt：第t期的自由现金流
  • r：折现率（加权平均资本成本WACC）
  • t：期数

自由现金流计算 ​

自由现金流（FCF）= 经营现金流 - 资本支出

详细计算：

FCF = 净利润 + 折旧摊销 - 营运资本增加 - 资本支出

折现率确定 ​

加权平均资本成本（WACC）：

WACC = (E/V)×Re + (D/V)×Rd×(1-T)

其中：

  • E：股权价值
  • D：债务价值
  • V：企业总价值
  • Re：股权成本
  • Rd：债务成本
  • T：企业所得税率

股权成本（CAPM模型）：

Re = Rf + β×(Rm - Rf)

增长率预测 ​

两阶段增长模型：

  • 前N年：详细预测期（通常5-10年）
  • N年后：永续增长期

永续增长率考虑因素：

  • GDP增长率
  • 行业增长率
  • 企业竞争优势
  • 通胀预期

股息折现模型（DDM） ​

适用条件 ​

  • 有稳定分红政策
  • 成熟型企业
  • 收益稳定可预测

模型类型 ​

零增长模型：

P = D / r

稳定增长模型：

P = D1 / (r - g)

两阶段增长模型：

  • 高增长期（Gordon增长）
  • 稳定增长期

资产基础法 ​

净资产价值法 ​

原理：

企业价值 = 资产公允价值 - 负债公允价值

适用情况：

  • 资产密集型企业
  • 重工业企业
  • 房地产企业

重置成本法 ​

原理：
企业价值等于重新建造相同企业所需成本

考虑因素：

  • 土地成本
  • 建筑物成本
  • 设备成本
  • 组织建设成本

期权定价模型 ​

实物期权法 ​

适用场景：

  • 高成长性企业
  • 科技企业
  • 具有灵活性的投资机会

期权类型：

  • 增长期权
  • 延迟期权
  • 放弃期权
  • 转换期权

绝对估值步骤 ​

1. 基础分析 ​

  • 了解商业模式
  • 分析竞争优势
  • 确定行业地位

2. 财务预测 ​

  • 收入预测
  • 成本费用预测
  • 资本支出预测
  • 营运资本预测

3. 参数设定 ​

  • 折现率确定
  • 增长率预测
  • 终值计算方法选择

4. 敏感性分析 ​

  • 关键假设变化对估值影响
  • 情景分析（乐观、中性、悲观）
  • 风险评估

5. 估值结论 ​

  • 内在价值区间
  • 投资建议
  • 风险提示

绝对估值优势 ​

理论优势 ​

  • 基于企业基本面
  • 不受市场情绪影响
  • 考虑时间价值
  • 适合长期投资

实用优势 ​

  • 帮助识别投资机会
  • 提供买卖参考
  • 控制投资风险
  • 支持投资决策

绝对估值局限 ​

预测难度 ​

  • 未来不确定性高
  • 假设影响大
  • 主观性较强

参数敏感性 ​

  • 折现率影响巨大
  • 增长率假设关键
  • 终值占比过高

适用限制 ​

  • 不适合成长初期企业
  • 周期性企业预测困难
  • 新兴行业缺乏历史数据

提高估值准确性 ​

基础工作 ​

  • 深入了解企业
  • 分析行业趋势
  • 研究竞争对手

预测改进 ​

  • 多情景分析
  • 概率加权
  • 专家意见

风险控制 ​

  • 保守假设
  • 安全边际
  • 定期修正

投资应用 ​

价值投资 ​

  • 寻找低估股票
  • 长期持有策略
  • 逆向投资思维

风险管理 ​

  • 估值下限保护
  • 分散投资原则
  • 动态调整策略

组合管理 ​

  • 基于估值配置
  • 再平衡策略
  • 业绩评估基准
