# 财务分析

财务分析是评估企业经营状况、盈利能力和财务风险的重要工具，是投资决策的基础。

## 财务报表分析
### 资产负债表
**核心分析要点：**
- **资产结构**：流动资产vs非流动资产
- **负债水平**：资产负债率、有息负债率
- **净资产质量**：净资产收益率、资产周转率
- **现金流状况**：货币资金占比、应收账款质量

**关键指标：**
```
资产负债率 = 总负债 / 总资产
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货) / 流动负债
```

### 利润表
**核心分析要点：**
- **收入质量**：收入增长率、收入构成
- **盈利能力**：毛利率、净利率、营业利润率
- **费用控制**：费用率变化、研发投入占比
- **利润增长**：利润增长率、利润质量

**关键指标：**
```
毛利率 = (营业收入 - 营业成本) / 营业收入
净利率 = 净利润 / 营业收入
ROE = 净利润 / 净资产
```

### 现金流量表
**核心分析要点：**
- **经营现金流**：主营业务造血能力
- **投资现金流**：扩张投资情况
- **筹资现金流**：融资和分红情况
- **现金流匹配**：现金流与利润的匹配度

**关键指标：**
```
经营现金流净额 / 营业收入
自由现金流 = 经营现金流 - 资本支出
现金含量 = 经营现金流 / 净利润
```

## 财务比率分析
### 盈利能力比率
**毛利率分析：**
- 反映产品定价能力和成本控制能力
- 同行业对比分析竞争力
- 趋势分析判断经营稳定性

**净利率分析：**
- 综合反映企业经营效率
- 费用控制能力的重要体现
- 行业特性影响显著

**ROE分析（杜邦分析）：**
```
ROE = 净利率 × 资产周转率 × 权益乘数
```
- **净利率**：盈利能力
- **资产周转率**：运营效率
- **权益乘数**：财务杠杆

### 偿债能力比率
**短期偿债能力：**
- 流动比率（理想值2.0）
- 速动比率（理想值1.0）
- 现金比率

**长期偿债能力：**
- 资产负债率
- 产权比率
- 利息保障倍数

### 运营能力比率
**存货管理：**
- 存货周转率
- 存货周转天数
- 库存结构分析

**应收账款管理：**
- 应收账款周转率
- 应收账款周转天数
- 坏账准备分析

**资产使用效率：**
- 总资产周转率
- 固定资产周转率
- 净资产周转率

### 成长能力比率
**收入增长：**
- 营业收入增长率
- 复合增长率（CAGR）
- 增长质量分析

**利润增长：**
- 净利润增长率
- 利润增长可持续性
- 利润来源分析

## 财务质量评估
### 收入质量
**确认标准：**
- 收入确认是否谨慎
- 关联交易影响
- 收入构成稳定性

**分析要点：**
- 现金收入占比
- 收入与经营现金流匹配度
- 客户集中度风险

### 利润质量
**核心指标：**
- 现金含量比率
- 非经常性损益占比
- 盈利持续性

**分析要点：**
- 利润来源的稳定性
- 会计政策的保守性
- 利润与现金流匹配度

### 资产质量
**资产结构：**
- 有形资产vs无形资产
- 流动性分析
- 资产减值风险

**分析要点：**
- 资产真实性
- 资产使用效率
- 资产估值合理性

## 行业财务特征
### 不同行业财务特点
**制造业：**
- 资产较重，周转率较低
- 存货和应收账款管理重要
- 固定成本占比较高

**服务业：**
- 资产较轻，固定资产少
- 人力成本占比较高
- 现金流一般较好

**零售业：**
- 存货周转率高
- 应付账款管理重要
- 现金流特征明显

### 行业对比分析
**横向对比：**
- 同行业关键指标对比
- 行业平均值和优秀值
- 竞争对手财务分析

**纵向对比：**
- 历史趋势分析
- 周期性特征
- 成长阶段特征

## 财务造假识别
### 常见造假手段
**收入造假：**
- 虚构收入
- 提前确认收入
- 关联交易非关联化

**成本费用造假：**
- 费用资本化
- 成本核算不当
- 费用隐藏

**资产造假：**
- 资产价值虚增
- 资产减值计提不足
- 无形资产虚假

### 识别方法
**财务指标异常：**
- 毛利率异常
- 费用率异常
- 资产周转率异常

**现金流分析：**
- 利润与现金流背离
- 经营现金流异常
- 投资现金流异常

**审计意见分析：**
- 非标准审计意见
- 审计师变更
- 内控缺陷披露

## 投资应用
### 基本面投资
- 选择财务稳健的企业
- 关注财务改善机会
- 规避财务风险高的企业

### 量化投资
- 建立财务指标筛选体系
- 多因子模型构建
- 回测验证策略有效性

### 风险控制
- 设置财务指标阈值
- 动态监控财务变化
- 及时调整投资组合
